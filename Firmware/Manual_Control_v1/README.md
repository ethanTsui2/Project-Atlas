# Manual Control v1

## Purpose

Initial firmware used to validate MG90S servo operation using a single 10k potentiometer.

## Hardware

- Arduino Uno
- MG90S Servo
- 10k Potentiometer

## Features

- Reads analog input from potentiometer.
- Maps input to servo angle.
- Controls one MG90S servo.

## Status

✅ Validated

## Future Improvements

- Support multiple servos
- External power supply
- Motion recording
- Playback mode
