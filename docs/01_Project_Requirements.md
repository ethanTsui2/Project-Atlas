# Project Requirements

## Project Objective

Design and build a modular 4-DOF desktop robotic manipulator using commercially available components and custom 3D-printed parts while documenting the complete engineering design process.

---

# Functional Requirements

| ID | Requirement | Target |
|----|------------|--------|
| FR-01 | Degrees of Freedom | 4 |
| FR-02 | Manual Control | Potentiometers |
| FR-03 | Motion Recording | Save multiple positions |
| FR-04 | Motion Playback | Replay saved sequence |
| FR-05 | User Interface | Buttons + LCD1602 |
| FR-06 | External Power | 5V 3A |

---

# Mechanical Requirements

| ID | Requirement | Target |
|----|------------|--------|
| MR-01 | Servo | MG90S |
| MR-02 | Fasteners | M3 preferred |
| MR-03 | Manufacturing | FDM 3D Printing |
| MR-04 | Easy Assembly | Yes |
| MR-05 | Easy Maintenance | Modular design |

---

# Design Goals

- Low cost
- Modular
- Repairable
- Easy to manufacture
- Educational
- Well documented
