# MG90S Reference Model

This folder contains the MG90S servo reference CAD model used for Project Atlas.

## Source

The initial MG90S CAD model was created by Nicolas Corona and downloaded from GrabCAD.

Model: MG90S Micro Servo Motor – High Detail CAD Model (Internal Gears & Motor Included)

Author: Nicolas Corona

Source: GrabCAD

## Verification

Critical dimensions were checked in SolidWorks and compared against the physical MG90S servo before using the model for mechanical design.

## Use in Project Atlas

This model is used as a reference component for designing servo mounts, joints, and mechanical clearances.

All custom robotic arm components designed for Project Atlas are original work.
